# Fastest Path To Paid Work

## Positioning

Do not lead as generic freelancers. Lead as a two-person specialist engineering cell contracting through Coilworks GmbH and backed by InteSpring credibility:

**Human-interactive hardware, medical-device documentation, motion control, RF/drone systems, PCB/power electronics, and AI-accelerated engineering documentation.**

That story is stronger than competing on commodity software marketplaces.

## Revenue Rules

- Break-even floor: EUR 100/hour.
- Normal target: EUR 120-140/hour.
- Specialist target: EUR 150-180/hour for medical compliance documentation, architecture/design documentation, RF/drone control, safety-critical test automation, and PCB/power electronics packages.
- Minimum engagement: 4 paid hours.
- Standard analysis package: 40 hours.
- Prototype package baseline: 160 hours, adjusted by complexity, PCB stackup, component count, hardware risk, and documentation depth.
- Avoid low-rate marketplaces unless the project is strategically useful, fixed-scope, fast, and review-building.

## Priority Offers

1. **Medical Device Documentation Sprint**
   - Buyer: medical-device startups, SMEs, engineering managers, quality/regulatory teams.
   - Deliverables: design history file support, architecture docs, requirements traceability, verification plan, risk-control documentation, AI-assisted document cleanup.
   - Rate: EUR 140-180/hour or fixed sprint.

2. **System Architecture And Design Documentation Sprint**
   - Buyer: hardware/software product teams, R&D leads, grant-funded engineering teams.
   - Deliverables: architecture diagrams, interface specs, design rationale, test strategy, supplier-ready technical package.
   - Rate: EUR 120-160/hour.

3. **RF / Drone / Control Systems Troubleshooting**
   - Buyer: UAV teams, robotics teams, defense suppliers, test labs, embedded product companies.
   - Deliverables: control loop analysis, flight/control integration support, telemetry/interfaces, RF power testing, MATLAB/Simulink validation.
   - Rate: EUR 120-180/hour.

4. **PCB / Power Electronics / Test Automation Support**
   - Buyer: electronics SMEs, medical/hearing-aid/testing companies, industrial hardware teams.
   - Deliverables: PCB design/review, Altium Designer and KiCad work, sensor and microcontroller electronics, wearable/medical hardware, product electronics integration, wiring/test bench design, LabVIEW/NI automation, HIL/HILS testing, EMC/EMI prep support.
   - Rate: EUR 110-150/hour.

5. **Human-Interactive Mechanical Systems Feasibility**
   - Buyer: medical-device, lifting-aid, wearable, exoskeleton, logistics, defense companies.
   - Deliverables: feasibility study, rapid prototype plan, kinematic analysis, supplier pathway.
   - Rate: EUR 130-180/hour.

## Daily Operating Rule

Every opportunity gets one of four statuses: `new`, `qualified`, `applied/contacted`, `follow_up_due`, `won`, `lost`, `parked`. If it is not in the tracker, it does not exist.
